ROZMIARY

width

Za pomocą właściwości width możemy zmienić szerokość danego elementu HTML.


div {
	width: 500px;
}

height

Za pomocą właściwości height możemy ustalić wysokość danego elementu HTML.


div {
	height: 350px;
}

min-width

Za pomocą właściwości min-width możemy ustalić minimalną szerokość danego elementu HTML.


div {
	min-width: 530px;
}

max-width

Za pomocą właściwości max-width możemy ustalić maksymalną szerokość danego elementu HTML.


div {
	max-width: 300px;
}

min-height

Za pomocą właściwości min-height możemy ustalić minimalną wysokość danego elementu HTML.


div {
	min-height: 200px;
}

max-height

Za pomocą właściwości min-height możemy ustalić minimalną wysokość danego elementu HTML.


div {
	max-height: 400px;
}
